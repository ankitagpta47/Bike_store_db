# Bike Sales Dashboard (Power BI)

## Overview

This project is a Power BI dashboard designed to analyze bike sales performance and customer insights. The report provides interactive visualizations to help understand revenue trends, customer distribution, order dates, and city-level sales performance.

The dashboard was built using Microsoft Power BI and includes multiple report pages with interactive filtering and data exploration features.

---

## Features

* Revenue analysis dashboard
* Customer insights and total customer metrics
* City-wise sales breakdown
* Order date trend analysis
* Interactive filters and visuals
* Clean and user-friendly Power BI design

---

## Dashboard Pages

### Page 1 – Sales Overview

Includes:

* Revenue KPI
* Total Customers KPI
* City analysis
* Order date trends

### Additional Pages

The report also contains supporting pages and tooltip interactions for deeper analysis.
